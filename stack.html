Stack Tecnológico del Proyecto

Asistente de Compras por Voz

Volver
Categoría Componente Tecnología / Herramienta Propósito Principal Justificación de Elección
Frontend y Voz
Aplicación Móvil Flutter (Dart) + Android Studio Desarrollo de la app móvil accesible, con interfaz optimizada para comandos por voz. Framework moderno y multiplataforma. Facilita la creación de interfaces consistentes y la integración con APIs nativas de accesibilidad.
Reconocimiento de Voz (STT) Android Speech API Conversión de la voz del usuario a texto para ser procesado por el NLU. Integración nativa con el sistema operativo, optimizada para el rendimiento en el dispositivo y sin costos adicionales.
Síntesis de Voz (TTS) Android Text-to-Speech API Proporciona las respuestas del asistente de forma audible y clara para el usuario. Solución robusta, nativa y personalizable por el usuario en su propio dispositivo, respetando sus preferencias de accesibilidad.
Backend e Inteligencia Artificial
Asistente Conversacional (NLU) Rasa Open Source Procesamiento de Lenguaje Natural para entender las intenciones del usuario y gestionar el diálogo. Open-source, alto grado de personalización para el español uruguayo y control total sobre el modelo y los datos.
API REST Java (Spring Boot) Backend centralizado que conecta la app, la base de datos y la lógica de negocio (usuarios, pedidos). Ecosistema maduro, robusto y escalable, ideal para la creación de APIs seguras y de alto rendimiento.
Base de Datos Google Cloud SQL (PostgreSQL) Almacenamiento persistente, escalable y seguro para productos, usuarios e historial de pedidos. Servicio gestionado que reduce la carga administrativa, con alta disponibilidad, backups automáticos y seguridad integrada.
Infraestructura y DevOps
Contenerización Docker & Docker Compose Estandarización de los entornos de desarrollo y producción para Rasa, backend y otros servicios. Garantiza la portabilidad, consistencia y reproducibilidad del sistema, simplificando el despliegue y las pruebas.
Despliegue en la Nube Google Cloud Run Servicio sin servidor (serverless) para ejecutar los contenedores en un entorno escalable y gestionado. Modelo de pago por uso, escalado automático a cero y gestión simplificada. Ideal para empezar y crecer sin grandes costos fijos.
CI/CD y Control de Versiones Git + GitHub (GitHub Actions) Automatización de las pruebas, construcción de contenedores y despliegues en la nube. Ecosistema integrado y estándar en la industria. Facilita la colaboración y asegura la calidad del código y los modelos.
Calidad y Seguridad
Testing del Modelo NLU Python + Tkinter + Rasa Test Creación de una app de escritorio para testeo de intents, precisión y confianza del modelo. Permite realizar pruebas de calidad del modelo de forma rápida y visual, generando métricas para su mejora continua.
Monitoreo y Observabilidad Prometheus + Grafana Recolección de métricas de rendimiento del bot y del backend. Visualización en dashboards. Estándar de facto para la observabilidad en sistemas nativos de la nube. Permite detectar problemas proactivamente.
Seguridad en Contenedores Wazuh / Falco Detección de intrusiones y comportamiento anómalo en los contenedores en tiempo de ejecución. Soluciones open-source potentes, alineadas con una arquitectura de contenedores para una defensa en profundidad.
Buenas Prácticas de Seguridad HTTPS, JWT, R8, Play Integrity API Aplicación de seguridad en capas: cifrado en tránsito, autenticación segura, ofuscación de código e integridad de la app. Sigue los estándares de la industria para proteger los datos del usuario y la integridad del sistema en todos los niveles.
