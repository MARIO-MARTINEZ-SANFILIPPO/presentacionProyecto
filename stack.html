Introducción al Stack Tecnológico

La elección de un stack tecnológico robusto y coherente es fundamental para el éxito de cualquier proyecto de software. A continuación, se detalla el conjunto de tecnologías, frameworks y herramientas que hemos seleccionado para construir el "Asistente de Compras por Voz". Cada componente ha sido elegido estratégicamente para cumplir con los objetivos del proyecto, priorizando la escalabilidad, la seguridad, la eficiencia en costos y, sobre todo, la capacidad de ofrecer una experiencia de usuario accesible y de alto rendimiento. Esta arquitectura nos permite construir una solución moderna, mantenible y preparada para el futuro.

Categoría Componente Tecnologías Tentativas / Herramientas Propósito Principal Justificación de Elección
Frontend y Voz
Frontend y Voz Aplicación Móvil Flutter (Dart)Jetpack Compose Desarrollo de la app móvil accesible, con interfaz optimizada para comandos por voz. Flutter: Framework moderno y multiplataforma.
Jetpack Compose: Moderno kit de herramientas de UI declarativo para desarrollo nativo en Android.
Frontend y Voz Reconocimiento de Voz (STT) Google Cloud Speech APIAndroid Speech API Conversión de la voz del usuario a texto para ser procesado por el NLU. Google Cloud: Máximo control de privacidad (opción de pago).
Android API: Integración nativa sin costos, pero con menor control de datos.
Frontend y Voz Síntesis de Voz (TTS) Android Text-to-Speech APIGoogle Cloud TTS API Proporciona las respuestas del asistente de forma audible y clara para el usuario. Android API: Solución robusta y nativa sin costo.
Google Cloud: Ofrece voces más naturales y opciones avanzadas (opción de pago).
Backend e Inteligencia Artificial
Backend & IA Asistente Conversacional (NLU) Rasa Open Source Procesamiento de Lenguaje Natural para entender las intenciones del usuario y gestionar el diálogo. Open-source, alto grado de personalización para el español uruguayo y control total sobre el modelo y los datos.
Backend & IA API REST Java (Spring Boot) Backend centralizado que conecta la app, la base de datos y la lógica de negocio. Ecosistema maduro, robusto y escalable, ideal para la creación de APIs seguras y de alto rendimiento.
Backend & IA Base de Datos Google Cloud SQL (PostgreSQL/MySQL) Almacenamiento persistente, escalable y seguro para productos, usuarios e historial de pedidos. Servicio gestionado que reduce la carga administrativa, con alta disponibilidad, backups y seguridad integrada.
Infraestructura y DevOps
Infraestructura & DevOps Contenerización Docker & Docker Compose Estandarización de los entornos de desarrollo y producción para todos los servicios. Garantiza la portabilidad, consistencia y reproducibilidad del sistema, simplificando el despliegue.
Infraestructura & DevOps Despliegue en la Nube Google Cloud Run Servicio serverless para ejecutar contenedores en un entorno escalable y gestionado. Modelo de pago por uso y escalado a cero. Ideal para empezar y crecer sin grandes costos fijos.
Infraestructura & DevOps CI/CD y Control de Versiones Git + GitHub Actions Construcción de contenedores y despliegues en la nube. Ecosistema integrado y estándar en la industria. Facilita la colaboración y la calidad del código.
Calidad y Seguridad
Calidad y Seguridad Testing del Modelo NLU Python + Tkinter + Rasa Test App de escritorio para testeo de intents, precisión y confianza del modelo. Permite realizar pruebas de calidad del modelo de forma rápida y visual para su mejora continua.
Calidad y Seguridad Monitoreo y Observabilidad Prometheus + GrafanaGoogle Cloud Operations Recolección de métricas de rendimiento y logs. Visualización en dashboards. Prometheus: Estándar open-source para métricas.
Google Cloud Ops: Alternativa nativa con menor carga operativa.
Calidad y Seguridad Panel Administrativo AngularReact / Vue.js Interfaz web para la gestión de usuarios, pedidos y configuración del sistema. Angular: Framework robusto ideal para aplicaciones de negocio.
React/Vue: Alternativas modernas y populares.
Calidad y Seguridad Buenas Prácticas de Seguridad HTTPS, JWT, R8, Play Integrity, IAM, Wazuh/Falco Aplicación de seguridad en capas: cifrado, autenticación, ofuscación, integridad, control de acceso y monitoreo. Sigue los estándares de la industria para proteger los datos del usuario y la integridad del sistema en todos los niveles.
