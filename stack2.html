Introducción al Stack Tecnológico

La elección de un stack tecnológico robusto y coherente es fundamental para el éxito de cualquier proyecto de software. A continuación, se detalla el conjunto de tecnologías, frameworks y herramientas que hemos seleccionado para construir el "Asistente de compras por Voz". Cada componente ha sido elegido estratégicamente para cumplir con los objetivos del proyecto, priorizando la escalabilidad, la seguridad, la eficiencia en costos y, sobre todo, la capacidad de ofrecer una experiencia de usuario accesible y de alto rendimiento. Esta arquitectura nos permite construir una solución moderna, mantenible y preparada para el futuro.

Categoría Componente Tecnologías / Herramientas Propósito Principal Justificación de Elección
Frontend y Voz
Frontend y Voz Aplicación Móvil Flutter (Dart) Desarrollo de la app móvil accesible, con interfaz optimizada para comandos por voz. Framework moderno y multiplataforma que permite un desarrollo rápido y consistente para Android e iOS.
Frontend y Voz Reconocimiento de Voz (STT) Google Cloud Speech API Conversión de la voz del usuario a texto para ser procesado por el NLU. Google Cloud: Máximo control de privacidad (opción de pago)
Frontend y Voz Síntesis de Voz (TTS) Google Cloud TTS API Proporciona las respuestas del asistente de forma audible y clara para el usuario. Google Cloud: Ofrece voces más naturales y opciones avanzadas (opción de pago).
Backend e Inteligencia Artificial
Backend & IA Asistente Conversacional (NLU) Rasa Open Source Procesamiento de Lenguaje Natural para entender las intenciones del usuario y gestionar el diálogo. Open-source, alto grado de personalización para el español uruguayo y control total sobre el modelo y los datos.
Backend & IA Captura de Fallbacks n8nGoogle SheetsGoogle Drive API Automatización de workflows para capturar fallbacks de Rasa y almacenarlos en Google Sheets para su análisis. n8n se conecta con las APIs de Google para registrar cada fallback en una fila de Google Sheets, facilitando el análisis y la mejora continua del modelo NLU.
Backend & IA API REST (Legacy) Java (Spring Boot) Backend centralizado que conecta la app, la base de datos y la lógica de negocio. Ecosistema maduro, robusto y escalable, ideal para la creación de APIs seguras y de alto rendimiento.
Backend & IA Base de Datos MySQL (Dockerized) Almacenamiento persistente para el BackOffice (Django), dockerizado en la misma imagen. Solución open-source robusta. Al estar dockerizada junto al BackOffice, se simplifica el despliegue y la gestión.
Infraestructura y DevOps
Infraestructura & DevOps API Gateway / Microservicios Python + Flask Servir como API principal o para microservicios, conectando el frontend con la lógica de negocio y otros servicios. Flask es un framework ligero, flexible y escalable, ideal para construir APIs de alto rendimiento en desarrollo y producción.
Infraestructura & DevOps Proxy Server (CORS) Node.js Servidor intermediario para gestionar las llamadas a las APIs desde el navegador y evitar problemas de CORS (Ambiente local). Solución ligera, de alto rendimiento y ampliamente utilizada para crear proxies y gestionar peticiones HTTP de manera eficiente.
Infraestructura & DevOps Webhook Ngrok Exponer servicios locales (ej. Rasa, Flask) a internet de forma segura para pruebas de webhooks y demos. Herramienta estándar para crear túneles seguros de forma rápida, esencial para probar integraciones con servicios externos.
Infraestructura & DevOps Contenerización y Orquestación Docker & Docker Compose Empaquetar todos los servicios (Rasa, Flask, Grafana, etc.) en una imagen unificada para estandarizar los entornos. Garantiza la portabilidad y consistencia del sistema. Docker Compose simplifica la gestión de múltiples servicios.
Infraestructura & DevOps Despliegue en la Nube Oracle Cloud Free Tier Hosting de la infraestructura y servicios en un entorno de nube sin costo inicial. Ofrece recursos "Always Free" suficientes para desplegar y operar la aplicación sin incurrir en costos, ideal para las etapas iniciales del proyecto.
Infraestructura & DevOps Gestión de APIs y Credenciales Google Cloud Console Creación del proyecto, gestión de credenciales OAuth para n8n y activación de APIs (STT, TTS, Drive, Sheets). Plataforma centralizada para administrar todos los servicios de Google Cloud, garantizando un control seguro y unificado del acceso a las APIs.
Infraestructura & DevOps CI/CD y Control de Versiones Git + GitHub ActionsJenkins Construcción de contenedores y despliegues automáticos en la nube. GitHub Actions: Ecosistema integrado.
Jenkins: Servidor open-source potente para flujos de CI/CD complejos.
Calidad y Seguridad
Calidad y Seguridad BackOffice / Panel de Gestión Django Interfaz web para la gestión de usuarios, análisis de conversaciones y configuración del sistema. Framework de alto nivel en Python que permite un desarrollo rápido y seguro, con un panel de administración auto-generado.
Calidad y Seguridad Testing del Modelo NLU Python + Tkinter + Rasa Test App de escritorio para testeo de intents, precisión y confianza del modelo. Permite realizar pruebas de calidad del modelo de forma rápida y visual para su mejora continua.
Calidad y Seguridad Monitoreo y Observabilidad Prometheus + GrafanaGoogle Cloud Operations Recolección de métricas de rendimiento y logs. Visualización en dashboards. Prometheus & Grafana: Estándar open-source para métricas, desplegado en la imagen Docker.
Google Cloud Ops: Alternativa nativa en la nube.
Calidad y Seguridad Buenas Prácticas de Seguridad HTTPS, JWT, R8, Play Integrity, IAM, Wazuh/Falco Aplicación de seguridad en capas: cifrado, autenticación, ofuscación, integridad, control de acceso y monitoreo. Sigue los estándares de la industria para proteger los datos del usuario y la integridad del sistema en todos los niveles.
